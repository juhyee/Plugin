Swiper 옵션 설정

  • 슬라이드 사이 여백

    spaceBetween:num

  • 한 슬라이드에 보여줄 개수

    slidesPerView:num

  • 센터모드

    centeredSlides:boolean

  • 자동슬라이드

    autoplay: {
      delay: (시간 설정),num
      disableOnInteraction: (스와이프 후 자동 재생)boolean
    }

  • 슬라이드 반복 여부

    loop:boolean

  • loop 사용 시 마지막 슬라이드에서 다음 슬라이드가 보여지지 않는 현상 수정

    loopAdditionalSlides:num

  • 요소의 개수를 슬라이드 하단에 나타내 주는 UI

    pagination: {
      el:(페이징 태그 클래스 설정), string
      clickable:(버튼 클릭 여부) boolean
      type:, (버튼 모양 결정)string
    }

  • 슬라이드 좌우 버튼

    navigation: {
      nextEl:(다음 버튼 클래스 설정), string
      prevEl:(이전 버튼 클래스 설정), string
    }

  • 슬라이드 넘길 때 위치 고정 여부

    freeMode:boolean

  • 현재 활성 슬라이드높이 맞게 높이조정

    autoHeight:boolean

  • 슬라이드 터치 저항 여부

    resistance:boolean

  • 버튼으로만 슬라이드 조작이 가능(false-스와이핑안됨)

    allowTouchMove:boolean

  • 슬라이드가 1개 일 때 pager, button 숨김 여부 설정

    watchOverflow:boolean

  • 슬라이드 시작 부분 여백

    slidesOffsetBefore:num
    slidesOffsetAfter:num

  • 스타일을 변경하거나(예: 숨기기/표시) 하위 요소를 수정(슬라이드 추가/제거)할 때마다 스위퍼가 업데이트(초기화)

    observer:boolean
    slidesOffsetAfter:boolean

변수에 담아서 활용 예시

var slideSetting = {
  slidesPerView : 'auto',
  spaceBetween : 6,
  loop : false,
}
const slide = new Swiper('.mySlider', slideSetting)

swiper 메소드

ex) mySlider.autoplay.start()

  • 해당 슬라이드로 이동

    .slideTo(index, speed, runCallbacks)

  • 이전 슬라이드로 이동

    .slidePrev(index, speed, runCallbacks)

  • 다음 슬라이드로 이동

    .slideNext(index, speed, runCallbacks)

  • 자동 재생 시작

    .autoplay.start();

  • 자동 재생 정지

    .autoplay.stop();

  • 슬라이드 제거

    .destroy();
